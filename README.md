# mobius-docker

## Usage
### :bulb: Running the app locally
1. Clone this repository
```terminal
$ git clone https://github.com/seunghwanly/mobius-docker.git
```
2. Change directory to mobius
```terminal
$ cd mobius
```
3. Get dependencies
```terminal
$ npm install
````
4. Run concurrently : mobius, nCube, mosquitto
```terminal
$ npm run dev
```
